NAME
====

L10N::TLH - Klingon localization of Raku

SYNOPSIS
========

```raku
use L10N::TLH;
jatlh "Hello World";
```

DESCRIPTION
===========

L10N::TLH contains the logic to provide a Klingon localization of the Raku Programming Language.
